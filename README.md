# Reid Poole

The source for [reidpoole.com](https://www.reidpoole.com), built with Next.js,
TypeScript, and CSS.

## Development

This project requires Node.js 22.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```

## Deployment

The production site is deployed through Vercel from the GitHub repository.
