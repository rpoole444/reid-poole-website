# Poole and the Gang Brand Assets

## Black-background exports

The production-ready source files are in `black-background/`:

- `horizontal-wordmark.png`: wide headers, banners, and project cards
- `stacked-logo.png`: square social posts and compact layouts
- `compact-framed-logo.png`: avatars, tiles, and merchandise marks
- `framed-logo.png`: posters and prominent feature graphics
- `round-badge.png`: stickers, badges, and circular placements
- `monogram.png`: small icons and social avatars

Optimized JPEG copies for the website live in
`public/images/projects/poole-and-the-gang/`.

The older `*-reference.png` files contain a baked checkerboard and are retained
only as design references. They should not be used on the website.
